# 🍰 Cakepool Inventory Management System

A desktop-based Inventory Management System developed using **C#**, **Windows Forms**, and **SQL Server** for managing bakery operations efficiently.

---

## 📌 Features

- 🔐 User Login
- 👥 User Management
- 📦 Product Management
- 🏷 Category Management
- 👤 Customer Management
- 🛒 Order Management
- 📊 Dashboard
- 🔑 Forgot Password (OTP Demo)
- 🗄 SQL Server Database Integration

---

## 🛠 Technologies Used

- C#
- Windows Forms (.NET Framework 4.7.2)
- SQL Server
- ADO.NET
- Visual Studio 2022

---

## 📂 Project Structure

- Login
- Dashboard
- Users
- Products
- Categories
- Customers
- Orders
- Forgot Password
- Change Password

---

## 🚀 How to Run

1. Clone the repository.
2. Restore the SQL Server database (`cakepoolDB`).
3. Update the SQL Server connection string if needed.
4. Build and run the project in Visual Studio.

---

## 🔐 Forgot Password

The original implementation uses Gmail SMTP for sending OTP emails.
For the GitHub version, the OTP generation flow is demonstrated without exposing sensitive Gmail credentials.
